### SOBRE MÍ

Actualmente curso el 8vo ciclo de Ingeniería de Software. Mi enfoque profesional se basa en construir sistemas eficientes aplicando buenas prácticas, código limpio y una arquitectura sólida mediante herramientas como C4 Model y Structurizr. 

A lo largo de mi formación y mi experiencia en operaciones de campo, he desarrollado un perfil versátil orientado a resolver problemas reales:

* **Desarrollo Web:** Construcción de bases robustas con Angular y C#, evolucionando hacia ecosistemas ágiles con Vue.js. Priorizo la experiencia de usuario (UX) mediante interfaces limpias y minimalistas.
* **Desarrollo Móvil:** Experiencia en el ciclo de vida completo de la aplicación. Inicié con Flutter y actualmente desarrollo con Kotlin, integrando pasarelas, geolocalización y persistencia de datos (SQLite/Firebase).
* **Backend & Automatización:** Dominio de Python para la optimización de flujos operativos (clasificación masiva de evidencias y generación de reportes). Implementación de microservicios en Java bajo el patrón Command/Service.
